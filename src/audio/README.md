# Audio Module

## Features
- Kết nối mic & loa camera
- Stream audio từ camera lên cloud
- Nhận audio từ mobile/web để phát
- Ghi âm khi có sự kiện
- Xử lý & cải thiện audio
- Đồng bộ cài đặt từ cloud
- Tối ưu real-time
